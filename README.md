# 온라인 서점
• 기본에 충실한 온라인서점 개인 프로젝트입니다.
# 사용기술
• Front End : HTML, CSS, JavaScript

• Back End : JAVA, Spring, MyBatis

• Database : Oracle
# 주요기능
• 회원가입 및 로그인

• 장바구니, 구매 기능

• 관리자 계정 상품관리
